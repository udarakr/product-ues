WSO2 DS Home

WSO2 Dashboard Server (DS) provides a tool set to create dashboards and visualizations for enterprise requirements. Unlike traditional solutions, DS exposes the power of the server to the dashboard or the app developer. The dashboards and microsites created via DS are stored as Jaggery apps, where developers have full flexibility to work with Jaggery APIs to retrieve, manipulate and compose data.

Create Dashboard

The Dashboard is an Executive Information System, which gives users a Bird's Eye view of their System or Business with use of gadgets created. Users can create their gadgets to meet their business requirements, and then let the different set users to view their dashboard.

Personalize your Dashboard

The users can personalize the dashboard based on their own interest, and change the position of the gadgets and settings how they currently look.

Secure your Dashboard

Security is important aspect of the dashboards, and restrict which users can see the Dashboards, pages, and gadgets. Also the dashboards can be logged in Single Sign on with any identity providers which faciltates users to use their own infrastructure.
